lineTransformation

Content copied to clipboard

A transformation is what happens to every line whenever the Hologram is Hologram.updated. This allows you to set placeholders on every update, colorize lines, use hex colors, or whatever you like.

Since 7/19/22, this uses a Component. The default is to use Adventure's Mini Message to turn the content into a Component
